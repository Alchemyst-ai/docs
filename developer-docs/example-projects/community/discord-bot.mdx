# Discord AI Chat Bot

A simple Discord bot powered by [Alchemyst AI SDK](https://docs.getalchemystai.com/integrations/sdk/typescript-sdk) and OpenAI. The bot stores conversation context and provides smart answers to usersâ€™ questions in direct messages.

---

## Features

- Responds to **Direct Messages (DMs)** only.
- Stores conversation context using Alchemyst AI.
- Generates intelligent responses with OpenAI models.
- Easily extendable for new features and commands.

---

## Prerequisites

- Node.js v20+
- Discord Bot Token ([create one here](https://discord.com/developers/applications))
- Alchemyst AI API Key
- OpenAI API Key
