## Getting Started

<Steps>
  <Step title="Get your API Key">
    Sign up at [Alchemyst Platform](https://platform.getalchemystai.com) and grab your secret key from the dashboard.
  </Step>
  <Step title="Install the SDK">
    ```bash
    npm install alchemyst-ai # or pip install alchemyst-ai
    ```
  </Step>
  <Step title="Add your first Context">
    Push data to your namespace to make it immediately available to your agents.
  </Step>
</Steps>
