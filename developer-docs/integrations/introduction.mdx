# Integrations Overview

Alchemyst's integrations enable you to seamlessly connect our context cloud with your existing AI development workflow. Whether you're building with popular frameworks like LangChain and LlamaIndex, using AI-powered development tools like Claude Desktop and VS Code, or working with modern AI SDKs, Alchemyst provides the context layer that makes your AI systems smarter and more capable.

## Why Use Alchemyst Integrations?

Alchemyst serves as the connective tissue of your AI stack, providing:

- **Unified Memory Layer**: Maintain persistent, context-aware memory across different tools and sessions
- **Real-time Context Access**: Enable your AI systems to access and share information instantly
- **Framework Flexibility**: Work with your preferred AI frameworks without vendor lock-in  
- **Scalable Architecture**: Build solutions that grow from prototype to production seamlessly

### Step 2: Authentication Setup

All Alchemyst integrations require an API key for authentication:

1. **Get your API key** from the [Alchemyst Dashboard](https://app.getalchemystai.com)
2. **Set environment variable**: `ALCHEMYST_AI_API_KEY=your_api_key_here`
3. **Configure your integration** using the specific setup guide for your chosen platform

<Note>
Store your API key securely using environment variables or secure credential management systems. Never commit API keys to version control.
</Note>

### Step 3: Basic Usage Example

Here's a minimal example using our Python SDK:

```python
import os
from alchemyst_ai import AlchemystAI

# Initialize client
client = AlchemystAI(
    api_key=os.environ.get("ALCHEMYST_AI_API_KEY")
)

# Add context
response = client.v1.context.add(
    documents=[{
        "content": "Your contextual information here",
        "metadata": {"source": "user_upload"}
    }],
    source="my_app",
    context_type="resource"
)

# Search context  
results = client.v1.context.search(
    query="What information do I need?",
    similarity_threshold=0.8
)
```
